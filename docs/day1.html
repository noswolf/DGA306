🆚 return vs print — ต่างกันอย่างไร?
✅ return — ส่งค่ากลับ
def double(n): return n * 2 result = double(5) print(result) # 10
• ส่งค่ากลับให้ผู้เรียกใช้ฟังก์ชัน
• นำไปใช้ต่อได้ (เก็บใน variable)
• ไม่แสดงบนจออัตโนมัติ
🖨️ print — แสดงบนจอ
def show(n): print(n * 2) result = show(5) # แสดง 10 print(result) # None !
• แสดงข้อความบนหน้าจอเท่านั้น
• ฟังก์ชันที่ใช้ print แต่ไม่ return จะคืนค่า None
• ใช้สำหรับ debug หรือแสดงผลสุดท้าย
⚠️ ข้อผิดพลาดที่พบบ่อย: เขียน print() ใน function แต่ลืม return แล้วแปลกใจว่าทำไม variable ได้ค่า None

🔬 PCA — การลดมิติข้อมูล
PCA ช่วย "บีบ" ข้อมูลหลาย dimension ให้เหลือน้อยมิติ แต่ยังรักษารูปแบบข้อมูลไว้
📌 Principal Component Analysis (PCA) เปลี่ยนแกนของข้อมูลให้ PC1 เก็บ "ความแปรปรวน" ไว้มากที่สุด — เหมือนหาทิศทางที่ข้อมูล "กระจาย" มากที่สุด
✅ ใช้ PCA เมื่อ...
• มีตัวแปรเยอะ (10+ features)
• ต้องการ visualize ใน 2D/3D
• ตัวแปรมี multicollinearity
• ก่อนทำ clustering
⚠️ ข้อควรระวัง
• ต้อง standardize ข้อมูลก่อน
• PC ไม่มีความหมาย "ตรงๆ"
• สูญเสียข้อมูลบางส่วนเสมอ
• ดู explained variance ratio
🎯 K-Means Clustering
K-Means แบ่งข้อมูลออกเป็น k กลุ่ม โดยจัดแต่ละจุดให้อยู่ในกลุ่มที่ centroid ใกล้ที่สุด
📌 K-Means เลือก centroid แบบสุ่ม → กำหนดกลุ่มให้แต่ละจุด → ย้าย centroid → ทำซ้ำจนกว่าจะไม่เปลี่ยน
จำนวนกลุ่ม k = 3
กด ➡️ เพื่อเริ่ม
🔄 ขั้นตอน K-Means
1️⃣ สุ่มเลือก k centroids
2️⃣ assign แต่ละ point ให้ centroid ใกล้ที่สุด
3️⃣ คำนวณ centroid ใหม่ (mean ของ cluster)
4️⃣ วนซ้ำ 2-3 จนไม่มีการเปลี่ยน
